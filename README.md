- "vue": "^2.5.2"

- "element-ui": "^2.0.5",
具体请参考 https://github.com/wangyuanjun008/wyj-vue-security/blob/master/package.json

### 项目运行
- git clone https://github.com/wangyuanjun008/wyj-vue-security.git
- cd wyj-vue-security
- npm install
- npm run dev
